# ⚔️ Offensive Security

Offensive Security is about finding and exploiting vulnerabilities in systems. By understanding how attackers work, you can defend against them.

**Remember**: Only test systems you own or have explicit written permission to test.

## What You'll Learn

The typical attack follows a pattern:

1. **Reconnaissance**: Gather information about the target
2. **Scanning & Enumeration**: Discover services and vulnerabilities
3. **Exploitation**: Gain unauthorized access
4. **Post-Exploitation**: Maintain access and extract data
5. **Web Security**: Specific attacks on web applications

## The Attack Chain

```
Reconnaissance
    ↓
Scanning & Enumeration
    ↓
Exploitation
    ↓
Post-Exploitation
    ↓
Data Extraction/Persistence
```

## Important Reminders

- ✅ Test on systems you own
- ✅ Get written permission
- ✅ Use isolated lab environments
- ✅ Follow responsible disclosure
- ❌ Never attack without authorization
- ❌ Respect privacy
- ❌ Don't cause damage
