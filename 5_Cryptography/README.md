# 🔐 Cryptography

Cryptography is the science of securing information through encryption and other mathematical techniques.

2. [Hashing](02_hashing.md) - One-way encryption for data integrity

4. [SSL/TLS](04_ssl_tls.md) - Securing web traffic

## Core Concepts

- **Encryption**: Transform data into unreadable form
- **Decryption**: Transform encrypted data back to readable form
- **Key**: Secret used for encryption/decryption
- **Cipher**: Algorithm for encryption/decryption
- **Hash**: One-way function for data integrity

## Why Cryptography Matters

- **Confidentiality**: Encrypted data can't be read by others
- **Integrity**: Hashing detects if data was modified
- **Authentication**: Verify someone is who they claim
- **Non-repudiation**: Prove who did something

## Real-World Examples

- **HTTPS**: Encrypts web traffic
- **Password storage**: Hashing passwords
- **Digital signatures**: Proving authenticity
- **VPNs**: Encrypting entire connections
- **WhatsApp**: End-to-end encryption
