# 🏗️ Foundations: Build Your Cybersecurity Base

Welcome to the Foundations section! Before you can understand how to attack or defend systems, you need to understand **how systems actually work**.

Think of this like learning to drive—you need to understand how a car works before you can learn racing tricks or defensive driving techniques.

## What You'll Learn

### [Operating Systems Basics](01_operating_systems.md)
Understand what an operating system is and how Linux and Windows work. This is critical because most cybersecurity work happens on these systems.

### [Networking Basics](02_networking.md)
Learn how computers talk to each other. Networks are the backbone of everything in cybersecurity—you can't attack or defend what you don't understand.

### [Command Line Essentials](03_command_line.md)
Mastery of the terminal is essential for any cybersecurity professional. Learn bash, PowerShell, and command-line tools.

### [Programming Basics](04_programming.md)
Why you need to understand programming concepts (you don't need to be a developer, but understanding code helps tremendously).

## Learning Path

1. Start with **Operating Systems** - understand the systems you'll be working with
2. Move to **Networking** - understand how they communicate
3. Learn **Command Line** - tools to interact with systems
4. Understand **Programming** - concepts you'll encounter

## Key Principles

✅ **In-depth**: We explain the "why", not just the "what"

✅ **Practical**: Examples and hands-on exercises for each topic

✅ **Beginner-friendly**: No assumed knowledge
