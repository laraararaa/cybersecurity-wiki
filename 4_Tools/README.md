# 🛠️ Tools & Technologies

This section covers the key tools used in cybersecurity, both offensive and defensive.

## Tools Covered

1. [Kali Linux](01_kali_linux.md) - The penetration testing operating system
2. [Nmap](02_nmap.md) - Network scanning and enumeration
3. [Metasploit](03_metasploit.md) - Exploitation framework
4. [Wireshark](04_wireshark.md) - Network traffic analysis
5. [Burp Suite](05_burp_suite.md) - Web application testing

## Tool Categories

### Reconnaissance & Scanning
- Nmap - Network scanning
- Shodan - Search for exposed devices
- theHarvester - Email and subdomain harvesting

### Exploitation
- Metasploit - Exploitation framework
- SQLmap - SQL injection testing
- Burp Suite - Web penetration testing

### Post-Exploitation
- Meterpreter - Post-exploitation payload
- Empire - Post-exploitation framework

### Defensive Tools
- Snort - Intrusion detection
- Zeek - Network monitoring
- Splunk - Log analysis
- OSSEC - Host-based intrusion detection

### Forensics & Analysis
- Wireshark - Network packet analysis
- Volatility - Memory forensics
- The Sleuth Kit - File system forensics

---

## Important Reminders

- **These tools are powerful**: They can be used for good or evil
- **With permission only**: Only use on systems you own or have authorization
- **Learning environments**: Practice in isolated labs first
- **Ethical use**: These tools should be used responsibly
