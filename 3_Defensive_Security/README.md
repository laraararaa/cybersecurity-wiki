# 🛡️ Defensive Security

Defensive Security is about protecting systems from attacks. By understanding how attacks work (offense), you can build better defenses.

## What You'll Learn

The defensive strategy follows this approach:

1. **Security Principles**: Understand core defensive concepts
2. **Hardening Systems**: Make systems resistant to attack
3. **Access Control**: Control who can access what
4. **Monitoring & Detection**: Spot attacks in progress
5. **Incident Response**: Handle security breaches

## The Defense Cycle

```
Prevention (Hardening)
    ↓
Detection (Monitoring)
    ↓
Response (Incident Management)
    ↓
Recovery (Return to normal)
    ↓
Lessons Learned (Improve)
    ↓
[Back to Prevention]
```

## Core Principles

- **Confidentiality**: Only authorized people see data
- **Integrity**: Data isn't modified without permission
- **Availability**: Systems and data are accessible when needed
- **Authenticity**: Verify someone is who they claim
- **Non-repudiation**: Can't deny you did something
